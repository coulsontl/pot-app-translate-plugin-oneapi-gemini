# pot-app.translate.oneapi.gemini

OneApi中转之后的Gemini翻译插件 - 一个为pot-app开发的翻译插件

## 简介
这是一个基于One API的翻译插件，支持多语言翻译，提供高质量的翻译结果。

## 前置要求
- [pot-app](https://github.com/pot-app/pot-app) - 请先下载并安装pot-app

## 功能特点
- 支持多种语言之间的互译
- 提供专业、流畅的翻译结果

## 安装方法
1. 在pot-app中点击"偏好设置"
2. 选择"服务设置"
3. 点击"翻译-添加外部插件"
4. 选择本插件的发布包进行安装

## 注意事项
- 请妥善保管你的API密钥
- 翻译服务需要联网使用
